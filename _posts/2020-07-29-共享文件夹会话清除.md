共享文件夹会话清除.md



我们访问带密码保护的共享文件夹时第一次往往需要输入密码，但第二次访问时就不需要密码了，因为密码已经被系统记录了，用户信息

被保存在系统缓存中，这是系统用来简化操作，但有时我们访问同一个文件夹时，我们可能会要用不同的用户来登陆，而且被系统缓存记录

密码这样也不安全的，要想彻底清除记录在Windows系统中的密码有两种方法：

1、在运行中输入： control userpasswords2



在弹出的对话框中选择“高级”,然后选择“密码管理”，把保存的 IP 连接删除即可。如下图：












2 、在命令提示符下运行： net     use  *  /d

把所有共享全部删除。这样下次再访问共享连接时就会提示输入用户名和密码了。（其中*号是通配符，代表所有的意思）



或者先查看有哪些共享，再指定特定共享名进行缓存信息删除，如

1.在命令提示符中输入：

net use

状态     本地      远程                             网络

-----------------------------------------------------------------------------------------------------------------

OK               \\192.168.1.99\音乐         Microsoft Windows Network

命令成功完成。

2.再进行删除

net  use   \\192.168.1.99\音乐     /delete



(  \\192.168.1.99为共享连接的IP地址，\音乐为共享文件夹的共享名）
